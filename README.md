# movieApp :movie_camera:
Created a simple movie list app powered by GraphQL and React

## To get started :computer:
1. Start the server
   - `cd movieQL`
   - `npm install`
   - `npm start`
  (to get out `cd ..`)
2. Start the client
   - `cd movieql-client`
   - `npm install`
   - `npm start`
  
API is provided by [YTS](https://yts.am/)
